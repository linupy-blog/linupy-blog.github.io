anaconda 在mac上的安装与配置

本文将介绍anaconda在mac上的安装方法,基本用法及如何在atom上使用anadonda的虚拟环境

1 anaconda在mac上的安装

方法一

  1. 进入anaconda的官网下载界面download
  2. 这里可以看到两个选项 : 命令行 和 图形界面 这里我们选择命令行 版进行下载
  3. 下载完成后, 按 cmd + t 打开终端 进入我们的下载目录: 
cd ~/Download/
bash ./Anaconda3.sh #执行我们刚才下载的文件,文件名要看你自己的
  1. 在等待一会后anaconda就安装好了

方法二

  1. 如果你安装了brew (没安装也没关系 执行ruby -e \"$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)\"进行安装)
  2. 执行brew search anaconda会找到这个路径: homebrew/cask/anaconda
  3. 执行brew install homebrew/cask/anaconda

## 2 anaconda的基本使用
1. 创建一个新用户

conda create --name name_you_want python=version_you_want package1 package2 

python版本和包是可选的.默认为python2.7
2. 现在执行```conda info --envs```
    现在应该看到系统有两个环境*root* 和*你刚才建的环境*
3. 可以使用```conda activate your_env```来激活环境
4. 安装包
直接使用pip 安装即可: `pip install packeage_name`
5. 如果想删除一个环境可以使用`conda remove envs_name`

## 3 将anaconda配置的环境应用到atom

1. 打开atom后,按`shift + cmd + p`搜索 文件init
2. 进入init.coffee后在最后一行添加

process.env.PATH = ['~/anaconda/bin/', process.env.PATH].join(':') ```

  1. 重启即可
